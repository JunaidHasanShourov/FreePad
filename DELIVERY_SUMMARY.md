## 🚀 How to Use FreePad

### Step 1: Start the Application
```bash
cd "c:\Users\black\Documents\P-Drive\WP-With-AI\FreePad"
npm run dev
```

Application opens at: http://localhost:3000

### Step 2: Create Projects
- **New Writing** - Start writing with formatting
- **New Drawing** - Start drawing with pen & eraser
- **Free Writing** - Unrestricted writing mode

### Step 3: Use Features
- **Writing:** Select fonts, sizes, colors, apply formatting
- **Drawing:** Choose pen/eraser, adjust size, draw on canvas
- **Settings:** Click ⚙️ for centralized settings
- **Export:** Click 📤 to export in 5 formats

### Step 4: Manage Projects
- **Search:** Find projects by name
- **Filter:** Filter by type
- **History:** View all past projects
- **Delete:** Remove unwanted projects
